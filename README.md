# OneLauncher [BETA]
A Minecraft Launcher designed to be customizable, free, and open-sourced.

Hello! Welcome to OneLauncher, a customizable, free, and open-source Minecraft Launcher. There isn't many open-source launchers out there, atleast that I know of. So, I decided to try and make one! I'll try to make it as error-free and as human-readable much as possible, so you can use it easily. Want to ask for changes/updates? Go Ahead!

# Current Features

- Offline Auth Account
- Vanilla Support

# Future Feature Goals:
- Customizable Launcher UI (movable buttons, widgets, etc.)
- Custom Themes
- Other Mod Loaders Support
- Built-in Mod Discovery Page (From 2 platforms)
